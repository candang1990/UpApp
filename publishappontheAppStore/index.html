Create Certificate

cái này bạn chỉ cần tạo 1 lần nó sử dụng lại được.

Create App ID.

Để tạo App ID bạn có bundle ID app. Hãy mở App bang Xcode và coppy nó.

Tiếp theo mở https://developer.apple.com tạo App ID


Đã hoành thành tạo AppID

Create Provisioning Distribute.

Để submit app bạ phải tạo provisioning Distribute cho riêng ứng dụng đó.

Đầu tiên chon Provisioning/Distribute/button"+".
Chọn App store / continue.
Chọn AppID vừa tạo của app cần Submit. Chọn Certificate / continue.
Nhập tên Provisioning / chọn Generate.

Build app với provisioning Distribute.

Để submit App bạnj phải build thành công với provisioning distribute

Đầu tiên bạ phải nhập account của bạn.

Tiếp theo vào build setting để import Provisioning Profile.

note: bạn phải tao provisioning Development Để test ứng dụng trước.
